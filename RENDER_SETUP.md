# 🚀 Render Professional Setup - Optimum Therapy PT System

## **Step-by-Step Deployment Guide**

### **1. Prerequisites Check** ✅
- [x] Render Professional account
- [x] GitHub repository ready
- [x] All code files created
- [x] Migration scripts prepared

### **2. Push Code to GitHub**

```bash
# Initialize git repository (if not already done)
git init

# Add all files
git add .

# Commit with descriptive message
git commit -m "Optimum Therapy PT Practice Management System - Production Ready"

# Add your GitHub repository as origin
git remote add origin https://github.com/YOUR_USERNAME/optimum-therapy.git

# Push to main branch
git push -u origin main
```

### **3. Create Services in Render Dashboard**

#### **A. PostgreSQL Database** (First - Required by other services)

1. **Go to Render Dashboard** → New → PostgreSQL
2. **Configuration:**
   - **Name:** `optimum-therapy-db`
   - **Database:** `optimum_therapy`
   - **User:** `optimum_therapy_user`
   - **Region:** `US East 1` (closest to Puerto Rico)
   - **Plan:** `Pro` (included in your Professional account)
   - **Version:** `15` (latest stable)

3. **Click "Create Database"**
4. **Save the connection details** (you'll need DATABASE_URL)

#### **B. Web Service** (Main Application)

1. **Go to Render Dashboard** → New → Web Service
2. **Connect Repository:**
   - Select your GitHub repository
   - Branch: `main`
   - Root Directory: leave blank

3. **Configuration:**
   - **Name:** `optimum-therapy-web`
   - **Region:** `US East 1`
   - **Branch:** `main`
   - **Build Command:** `npm install && npm run build`
   - **Start Command:** `npm run preview`
   - **Plan:** `Pro` (included in Professional)

4. **Environment Variables** (Add these in the Environment section):

```env
# Database Connection (Auto-generated by Render)
DATABASE_URL=postgresql://optimum_therapy_user:password@hostname:5432/optimum_therapy

# Application Security
VITE_ENCRYPTION_KEY=your_32_character_encryption_key_here
NODE_ENV=production

# Email Service (Resend)
VITE_RESEND_API_KEY=re_your_resend_api_key

# SMS Service (Twilio)
VITE_TWILIO_ACCOUNT_SID=AC_your_twilio_account_sid
VITE_TWILIO_AUTH_TOKEN=your_twilio_auth_token
VITE_TWILIO_PHONE_NUMBER=+1787XXXXXXX

# Facility Information
VITE_FACILITY_NAME=Optimum Therapy
VITE_FACILITY_ADDRESS=Edificio Roman Carr 107 km 1.1, Aguadilla PR 00603
VITE_FACILITY_PHONE=787-XXX-XXXX
VITE_FACILITY_NPI=1477089696

# Application Settings
VITE_SESSION_TIMEOUT_MINUTES=15
VITE_EOD_SYNC_TIME=18:00
```

5. **Advanced Settings:**
   - **Auto-Deploy:** ✅ Yes
   - **Health Check Path:** `/`

#### **C. Cron Jobs** (Automated Tasks)

**Email Reminders Cron Job:**
1. **New** → Cron Job
2. **Configuration:**
   - **Name:** `email-reminders`
   - **Command:** `node scripts/email-reminders.js`
   - **Schedule:** `0 18 * * *` (6 PM AST daily)
   - **Region:** `US East 1`

**End-of-Day Sync Cron Job:**
1. **New** → Cron Job  
2. **Configuration:**
   - **Name:** `eod-sync`
   - **Command:** `node scripts/eod-sync.js`
   - **Schedule:** `0 18 * * *` (6 PM AST daily)
   - **Region:** `US East 1`

